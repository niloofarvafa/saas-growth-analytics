# saas-growth-analytics
# SaaS Growth Analytics Framework

## 🎯 Objective
Design a complete analytics system to measure acquisition efficiency, retention performance and revenue growth in a SaaS environment.

---

## 📊 Key Metrics

- Customer Acquisition Cost (CAC)
- Funnel Conversion Rate (Visit → Lead → Customer)
- Cohort Retention Analysis
- Lifetime Value (LTV) Estimation
- Channel Efficiency Comparison

---

## 🛠 Tech Stack

Python (Pandas, NumPy)  
SQL  
Power BI  
Excel  

---

## 📈 Business Impact

This framework enables founders and growth teams to:

- Identify highest-performing acquisition channels  
- Detect funnel drop-offs  
- Reduce churn risk  
- Optimise marketing budget allocation  

---

## 📂 Project Structure

- `data/` → Raw & processed datasets  
- `sql/` → Data transformation queries  
- `notebooks/` → Analysis & modelling  
- `dashboard/` → KPI dashboard design
